Ik kreeg van Holdbaar de vraag een simpele website te bouwen wat diende als visite kaartje.

Hierbij vroeg Holdbaar specifiek voor het posten van projecten.

Uiteindelijk heb ik voor Holdbaar een one-page website gemaakt met:
- Visueel aantrekkelijke stijl
- Verschillende content secties
- Project slider
- Responsive design
- Basic SEO optimalisatie

Daarnaast heb ik geholpen met kleinere aanpassingen zoals:
- Het uitknippen en optilamiseren van het logo voor web 
- Maken van een hero sectie afbeelding
- Content vulling

Ik heb de website opgeleverd met een handleiding voor het aanpassen van content en het toevoegen en aanpassen van projecten.


Holdbaar vroeg mij een eenvoudige website te bouwen die als visitekaartje kon dienen, met een focus op het tonen van projecten. Ik heb een overzichtelijke one-page website met een visueel aantrekkelijke stijl, verschillende contentsecties, een projectslider, responsive design en basis SEO-optimalisatie opgeleverd.

Daarnaast heb ik Holdbaar ondersteund bij:

- Het uitknippen en optimaliseren van het logo voor web
- Het ontwerpen van een hero-afbeelding
- Het vullen van de content

Bij oplevering heb ik een handleiding toegevoegd zodat Holdbaar eenvoudig zelf content en projecten kan aanpassen en toevoegen.
